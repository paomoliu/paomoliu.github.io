获取iOS与Mac设备型号（标识符）

iOS设备型号（标识符）

+ (NSString *)machineName {
    struct utsname systemInfo;
    uname(&systemInfo);
    NSString *machineName = @"";
    machineName = [NSString stringWithCString:systemInfo.machine
                                     encoding:NSUTF8StringEncoding];
    return  machineName;
 }

Mac设备型号（标识符）

+ (NSString *)machineName:(NSString *)name {
    size_t size;
    sysctlbyname(name, NULL, &size, NULL, 0);
    char *machine = (char *)malloc(size);
    sysctlbyname(name, machine, &size, NULL, 0);
    NSString *machineName = [NSString stringWithCString:machine encoding:NSUTF8StringEncoding];
    free(machine);

    return machineName;
 }

使用sysctlbyname需添加头文件#include <sys/sysctl.h>；
对于该方法，iOS也可以使用该方式获取标识符，iOS传hw.machine，Mac传hw.model；

SourceTree成功访问SVN

个人比较喜欢使用SourceTree，以前从未使用它访问过SVN。由于实习，公司使用的SVN管理项目源码，因而有了SourceTree访问SVN的尝试。在这个过程中遇到很多的坑，所以做个总结，方便自己以后查看。

SourceTree简介

SourceTree是Windows和Mac OS下免费的git和hg客户端，同时也是Mercurial和Subversion版本控制系统工具。支持创建、克隆、提交、push、pull和合并等操作。

SourceTree安装

不建议直接从App Store搜索下载SourceTree，因为App Store里的版本太低。建议直接前往官网下载。安装步骤如下：

  1. 下载最新版SourceTree；
  2. 双击打开压缩包的.dmg文件，将SourceTree的App应用拖放到Applications里；
  3. 在Lanchpad中找到SourceTree并打开它；
  4. 接受协议，Continue；
  5. 设置指定的账户登录（我一般使用github帐号进行登录），也可以直接Skip该步骤；
  6. 安装成功。

git基础

在大一下的一门选修课上第一次接触到git，当时所有的操作都是在Linux系统下进行，觉得十分的深奥。在后面的学习中，偶尔会用一用一些基本的命令，最近发现关于git的一些内容都忘了一些，于是重新拾起当时老师讲的课件，做一个复习总结，也方便自己以后查阅。

本章介绍使用git前的基础知识，了解版本控制系统（VCS），git项目的三个区域。下一篇总结git的常见用法。

1. 什么是版本控制？

版本控制是一种用于跟踪文件版本的系统，可以记录若干文件的内容变化，以便将来查阅特定版本修订情况。我们可以对任何类型的文件进行版本控制。

如果你是位图形或网页设计师，可能会需要保存某一幅图片或页面布局文件的所有修订版本。采用版本控制系统是个明智的选择。有了它就可以将某个文件回溯到之前的状态，甚至将整个项目回退到过去某个时间点的状态。

许多人习惯用复制整个项目目录的方式来保存不同的版本，或许还会改名加上备份时间以示区别。这么做唯一的好处就是简单，不过坏处却不少：有时会混淆所在的工作目录，弄错了文件丢了数据就没了后退的路。

NSAttributedString与NSMutableAttributedString学习总结

在研究开源中国的源代码过程中，发现它里面的评论、点赞等按钮既有图又有文字，即所谓的图文混排，并不是我所想的UIImageView＋UIButton。里面还有一些属性字符串，支持链接、@等。这些都是利用DTCoreText 和TTTAttributedLabel 两个第三方库实现。于是想自己了解一下富文本相关的知识，开始学习该部分内容，做个小总结。这篇总结主要是参考官方文档。

NSAttributedString可以对字符串附加格式信息，比如字体和段落对齐。可以在整个字符串上使用这些元数据，也可以对字符串的不同部分使用不同的属性。其字符串是只读的。我们将字符与属性的关联称为属性字符串（attributed string）。NSMutableAttributedString是NSAttributedString的子类，它是可修改的属性字符串。
